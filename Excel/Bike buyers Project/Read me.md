
Bike Sales Analysis – Excel Dashboard

Overview

This project presents an interactive Bike Sales Dashboard built using Microsoft Excel.
The dashboard analyzes customer demographics and purchasing behavior to understand factors influencing bike purchases.

The objective of this project is to demonstrate data analysis, dashboard creation, and data visualization skills using Excel.

Dashboard Insights

The dashboard provides several key insights including:

- Average Income by Gender
  - Comparison of income levels between male and female customers and their bike purchasing behavior.

- Customer Commute Distance
  - Analysis of how commute distance influences bike purchases.

- Bike Purchases by Age Group
  - Visualization of purchasing trends among adolescents, middle-aged, and older customers.

- Customer Demographics Filters
  - Interactive slicers allow filtering data by:

  - Marital Status
  - Education Level
  - Region
