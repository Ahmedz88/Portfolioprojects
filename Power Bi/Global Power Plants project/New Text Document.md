
🌍 Global Power Plant Dashboard (Power BI)

📊 Overview

This project analyzes global power generation data to uncover insights into capacity distribution, fuel mix, and renewable energy trends.

---

🚀 Key Features

1. Interactive dashboard with slicers (Country, Fuel, Year)
2. Global capacity visualization using map
3. Top countries and plants analysis
4. Renewable vs non-renewable trends
5. Advanced analytics (Top 20 contribution, matrix heatmap)

---

📈 Dashboard Pages

1️. Executive Overview

1. KPIs (Total Capacity, Generation, Plants)
2. Global map visualization
3. Fuel mix distribution

2. Analysis

1. Capacity growth over time
2. Fuel vs country comparison
3. Renewable trends

3. Insights

1. Top 20 power plants
2. Capacity share analysis
3. Advanced matrix visualization

🎯 Key Insights

1. Fossil fuels still dominate global capacity
2. Renewable energy shows strong growth trend
3. A small number of plants contribute a large share of capacity
