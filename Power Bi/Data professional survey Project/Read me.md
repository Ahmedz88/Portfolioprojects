Data Professional Survey Analysis – Power BI Dashboard

Overview

This project presents an interactive dashboard analyzing results from a Data Professional Survey.
The dashboard explores salary trends, programming language preferences, job roles, and work-life balance among data professionals.

The goal of this project is to demonstrate data visualization, data modeling, and dashboard design skills using Power BI.

Dashboard Insights

The dashboard provides insights into:

- Average Salary by Job Title
   - Comparison of salaries among roles such as Data Scientist, Data Engineer, Data Analyst, and Data Architect.

- Country Distribution of Survey Participants
   - Visualization of survey respondents by country.

- Favorite Programming Languages
   - Shows which programming languages are most popular among data professionals.

- Average Age of Survey Participants

- Work-Life Balance vs Salary Happiness

- Gender Distribution of Respondents

- Difficulty of Breaking into the Data Field


Tools Used:

- Power BI Desktop
- Data Cleaning and Transformation (Power Query)
- Data Visualization and Dashboard Design

Key Skills Demonstrated:

- Data cleaning and transformation
- Data visualization
- Dashboard development
- Data storytelling
- Exploratory data analysis

Dataset:

The dataset contains responses from data professionals including:

- Job title
- Salary
- Country
- Gender
- Programming language preference
- Work-life balance satisfaction
- Difficulty entering the data field

Dashboard Preview:

The dashboard includes multiple visualizations such as:

- Treemap (Country distribution)
- Bar chart (Average salary by job title)
- Donut charts (Gender and difficulty entering data field)
- Gauge charts (Work-life balance and salary happiness)
