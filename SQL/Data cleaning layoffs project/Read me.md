Project
Layoffs Data Cleaning & Analysis

This project focuses on preparing a raw layoffs dataset for analysis.

Key tasks performed:
- Removed duplicate records
- Handled missing and null values
- Standardized company, industry, and country names
- Converted and formatted date fields
- Conducted exploratory analysis to identify layoffs trends

Insights explored:
- Layoffs by company
- Layoffs by industry
- Layoffs by country
- Layoffs trends over time
