Airbnb Pricing Analysis – Tableau Dashboard

This repository contains a Tableau data visualization project that analyzes Airbnb listing prices in Seattle. The dashboard explores how average prices vary by zipcode, bedroom count, and listing year, helping identify pricing patterns across locations and property sizes.

Project Overview

The main goals of this project are to:

- Visualize the average Airbnb listing price by zipcode
- Analyze how price changes with the number of bedrooms
- Examine price trends over time
- Present insights through interactive Tableau dashboards

All visualizations were created using Tableau Desktop Public Edition and can be published to **Tableau Public.


Dataset

- Source: Airbnb dataset
- Format:Excel (.xlsx)

Key Fields

- zipcode – Zip code of the listing location  
- price – Nightly listing price  
- bedrooms – Number of bedrooms in the property  
- date / year – Used to derive yearly trends

Dashboard Components

The Tableau dashboard includes the following visualizations:

1. Avg Price by Zipcode (Map)
- Geographic map of Seattle zipcodes
- Colored based on **average listing price

2. Avg Price by Zipcode (Bar Chart)
- Bar chart ranking zipcodes by average price
- Highlights the most and least expensive areas

3. Avg Price per Bedroom (Bar Chart)
- Shows how average price varies by number of bedrooms

4. Avg Price per Bedroom (Table)
- Detailed table showing:
  - Bedroom count
  - Number of listings
  - Average price

5. Avg Price by Year (Bubble Chart)
- Each bubble represents a year
- Bubble size/color indicates average listing price
